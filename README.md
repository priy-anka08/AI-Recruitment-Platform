# AI Recruitment & Workforce Management Platform

## Project Overview

AI-powered Recruitment & Workforce Management Platform developed using FastAPI, React, PostgreSQL, and Gemini AI. The system automates recruitment, resume screening, interview scheduling, candidate tracking, project management, sprint planning, and analytics.

## Features

* JWT Authentication & Role-Based Access Control
* Job Management System
* AI Resume Screening & ATS Scoring
* AI Voice Screening Agent
* Interview Scheduling & Calendar Integration
* Candidate Tracking System
* Project & Team Management
* AI Task Generation
* Developer Assignment
* Sprint Planning
* Analytics Dashboard

## Tech Stack
